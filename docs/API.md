# API Documentation

## Functions

### greet(name)
Returns a personalized greeting message.

**Parameters:**
- `name` (str): The name to greet

**Returns:**
- str: A greeting message

### calculate_sum(a, b)
Calculates the sum of two numbers.

**Parameters:**
- `a` (int/float): First number
- `b` (int/float): Second number

**Returns:**
- int/float: The sum of a and b
